# Modulo Messaging

## Responsabilidade
Centraliza templates de mensagens operacionais por tipo e canal.

## Escopo funcional
- Cadastro/versionamento de templates.
- Selecao de template por tipo (`TemplateTypeChoices`) e canal (`ChannelChoices`).
- Base para envio de notificacoes disparadas por tools do assistente.
