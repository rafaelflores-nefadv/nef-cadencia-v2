# Modulo Integrations

## Responsabilidade
Centraliza configuracoes de conectores externos por canal de notificacao.

## Escopo funcional
- armazenar configuracoes JSON por canal (`Integration`)
- habilitar/desabilitar conectores
- servir como backend para dispatch em `assistant.tools_actions`
