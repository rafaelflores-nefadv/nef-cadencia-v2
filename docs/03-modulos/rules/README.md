# Modulo Rules

## Responsabilidade
Fornece configuracao dinamica do sistema sem necessidade de deploy.

## Escopo funcional
- Armazenar parametros em `SystemConfig`.
- Expor leitura tipada via `rules.services.system_config`.
- Permitir que outros modulos ajustem comportamento em runtime.
